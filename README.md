# 스켈레톤 로딩화면 구현

API를 통해 통신을 받아오기 전 띄우는 화면


사용자들이 화면을 통해 속도가 빠르다고 느껴지도록 함


ex. 유튜브에서 썸네일이 로딩되기 전에 회색배경이 먼저 보이게 함



# 프로젝트 시작 방법
## npm 설치
`npm install`

## npm 시작
 `npm start`




## 구현화면
- 왼쪽은 스켈레톤을 사용하지 않고 api를 받아오는 화면, 
- 오른쪽은 스켈레톤을 사용한 후, api를 통해 받아온 데이터를 출력하는 화면이다.
